Shopping Portal

A comprehensive shopping portal built with React.js, Node.js, and SQL Server.

Features

  • Dynamic product listings with real-time search and filtering.
  • User authentication and secure payment processing.
  • Responsive design for seamless user experience across devices.

Technologies Used

React.js, Node.js, Express, SQL Server, RESTful APIs.

Challenges & Learnings

Optimized backend performance, implemented robust security protocols, and enhanced frontend responsiveness.
